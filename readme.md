# TIA

## Auteur

William STEPHAN

## Fonctionnalités envisagées

- Géneration de chemin aléatoire
- Plusieurs types de tours
- Plusieurs types d'enemis
- Systeme de vague
- Systeme de vie et d'argent

## Fonctionnalités développées

- 3 types de tours (Minigun, Canon, Rocket)
- 3 types d'enemis (Light, Medium, Heavy)
- Systeme de vague (10 vagues)
- Systeme de vie et d'argent

J'ai tenté d'implémenter une génération de chemin aléatoire mais cela était assez dur car les chemin généré n'étais pas pratique pour le jeu. C'était la plus part du temps des lignes droite, ou juste un virage simple.

## Manuel d'utilisation

C'est un Tower Defence classique, des vagues d'enemis vont se lancer sur le chemin pour aller détruire votre base, votre objectif est de tuer ces enemis avant qu'ils atteingent votre base. Un enemi qui atteint votre base vous fait perdre une vie, vous en avez dix. Pour vous défendre, vous pouvez placer des tours le long du chemin qui attaquerons automatiquement les enemis proches. Il y a 3 types de tours qui ont un prix différents et des fonctionnements différents. Un enemis tué vous donne un quantité d'argent, en fonction du type d'enemi tué.

Pour placer une tourelle, cliquer avec votre doigt sur une case, si elle devient violette, c'est qu'elle est bien sélectionnée. Puis choisissez le type de tour que vous voulez placer.
